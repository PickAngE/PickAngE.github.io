Quick Start Guide

1. Invite the Bot

Add Kernel Bot to your Discord server with the necessary permissions.

2. Initial Setup

Use /adminhelp to see all available configuration commands.

3. Configure Logs

Use /logconfig #your-channel to enable moderation logs.

4. Customize Messages

Configure welcome messages with /welcome and leaving messages with /leaving.

5. Enable Tickets

Configure the ticket system with /ticket setup.

6. Your Turn

Once configured, you can now use the commands to manage your server.

Important Information

Security and Validation

Kernel Bot integrates advanced security measures:

  • User input validation
  • Rate limiting per command
  • Permission verification before execution
  • Secure error messages

Support and Updates

The bot is regularly updated with new features and bug fixes.

  • Regular updates
