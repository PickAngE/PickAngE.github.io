Kernel Bot Documentation

Bot Commands

Here you will find a list of all available commands, grouped by category.

Support Commands

Ticket System

Creates a new support ticket to help users with their issues privately.

/ticket setup #category @support_role #archive_channel @mention_support

Logs Commands

setlog

Sets the channel used for the bot's logs.

/setlog #channel

Showlog Config

Displays the server's log configuration.

/showlogconfig

Logconfig

Easily configure server logs.

/Logconfig

Utility Commands

User Info

Displays detailed information about a specific user in the server.

/Info @user

Reaction Role

Sets up a reaction role on a message.

/reactionrole messge_id emoji @role

Mention Role

Configures automatic user mentions on channel creation.

/mentionuser adduser @user

Voice Chat Commands

Move

Moves a user to another voice or stage channel.

/move @user

Moveall

Moves all users from one voice/stage channel to another.

/moveall #voice_chat_1 #voice_chat_2

Moveall2

Moves all connected voice/stage members to the specified channel.

/moveall2 #voice_chat

Help Commands

Help

Displays the list of commands.

/help

Adminhelp

Displays the list of commands reserved for admin.

/adminhelp
